🍍 Malee Platform — Customer Journey

📊

Type A: Dashboard & Analytics

👤 ผู้บริหาร / Sales Manager / Marketing
1
🌐
เข้าสู่ระบบ Malee Platform
เปิดเบราว์เซอร์ → Login ด้วย Account ของบริษัท
2
📈
ดู Overview Dashboard
ติดตามยอดขายรวม, KPI Cards, Trend รายเดือน
Filter ตาม Product Group / Region / ช่วงเวลา
3
🔍
EDA / Accuracy Deep Dive
วิเคราะห์ WAPE, Bias ตาม SKU / ภูมิภาค
ดู Heatmap, Rankings, แนวโน้มความแม่นยำ
✓
✅
ตัดสินใจ & Action
ปรับ Stock / วางแผน Marketing / รายงานผู้บริหาร
Export เป็น Report ได้ทันที
🤖

Type B: Full AI Forecast

👤 Data Team / Demand Planner / Supply Chain
1
🌐
เข้าสู่ระบบ Malee Platform
เปิดเบราว์เซอร์ → Login ด้วย Account ของบริษัท
2
📈
ดู Overview Dashboard
ดู Baseline ยอดขายปัจจุบัน
เข้าใจ Trend ก่อนรัน Forecast
3
📁
Upload ข้อมูลยอดขายใหม่
อัปโหลดไฟล์ CSV ที่มีข้อมูลยอดขายล่าสุด
ระบบ Validate ให้อัตโนมัติ
4
🤖
รัน AI Forecast
กดปุ่ม Run → AI ประมวลผลด้วย LightGBM + Prophet
รอ ~40 วินาที → ได้ผลลัพธ์ทันที
5
🎯
วิเคราะห์ผล & Scenario Planner
ดู Forecast Chart + จำลอง Promotion
ปรับ Discount / Duration → เห็นผลทันที
✓
✅
ตัดสินใจ & Action
วางแผนการผลิต / Promotion Campaign
ลด Overstock & Stockout ด้วยข้อมูล
💡 Sales Tip: ลูกค้าที่เริ่มจาก Type A สามารถ upgrade เป็น Type B ได้ภายหลัง — ขายเป็น 2 Tier ช่วยปิด deal ได้ง่ายขึ้น!

📊 เล่า Type A — Dashboard & Analytics

ใช้กับลูกค้าที่มี Data Team อยู่แล้ว หรือต้องการแค่ Visualization

"ระบบของเราช่วยให้ผู้บริหารเห็น ยอดขาย, KPI, แนวโน้ม ได้ครบจบในหน้าเดียว

Filter ได้ตาม Product / Region / ช่วงเวลา → ไม่ต้องเปิด Excel อีกต่อไป

มี EDA Dashboard ที่วิเคราะห์ให้เลยว่า สินค้าไหนแม่น สินค้าไหนเบี่ยงเบน → ตัดสินใจได้เร็วขึ้น"

🤖 เล่า Type B — Full AI Forecast

ใช้กับลูกค้าที่ต้องการ Demand Planning / Supply Chain Optimization

"นอกจาก Dashboard แล้ว ระบบยังมี AI Forecast ที่ทำนายยอดขายล่วงหน้าได้

แค่อัปโหลดไฟล์ → กดปุ่ม → ได้ผลใน 40 วินาที ไม่ต้องเขียน Code เลย

ยังมี Scenario Planner จำลองว่า ถ้าลด 10% จะขายเพิ่มเท่าไหร่ → วางแผน Promotion ได้แม่นยำ"
