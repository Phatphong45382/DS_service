# Purchase Order (PO) Template Specification

ใช้ spec นี้ให้ AI Agent สร้างภาพ PO จำลองสำหรับทดสอบระบบ AI OCR
(ระบบจะอ่านภาพแล้วดึงข้อมูลออกมาเป็น JSON อัตโนมัติ)

---

## JSON Schema ที่ระบบ OCR คาดหวัง

```json
{
  "po_number": "string — เลขที่ PO",
  "po_date": "string — DD/MM/YYYY",
  "customer_name": "string — ชื่อบริษัทผู้สั่งซื้อ",
  "customer_address": "string — ที่อยู่ (ถ้ามี)",
  "delivery_date": "string — DD/MM/YYYY (ถ้ามี)",
  "items": [
    {
      "line_no": "number",
      "product_code": "string — รหัสสินค้า",
      "product_name": "string — ชื่อสินค้า (ชื่อเต็ม flavor + size)",
      "quantity": "number — จำนวน",
      "unit": "string — หน่วย เช่น ลัง, แพ็ค, กล่อง",
      "unit_price": "number — ราคาต่อหน่วย (บาท)",
      "total_price": "number — quantity × unit_price"
    }
  ],
  "subtotal": "number — รวมก่อน VAT",
  "vat": "number — VAT 7%",
  "grand_total": "number — รวมทั้งสิ้น",
  "notes": "string — หมายเหตุ (ถ้ามี)"
}
```

## คำสั่งสำหรับ AI Agent สร้างภาพ

ใช้ prompt นี้:

> สร้างภาพ Purchase Order (ใบสั่งซื้อ) ตามข้อมูลด้านล่าง
> - ให้เป็นเอกสารที่ดูสมจริง มีเส้นตาราง, header, logo placeholder
> - ใช้ฟอนต์ภาษาไทยชัดเจน
> - ขนาดภาพ ~A4 (794×1123px หรือใกล้เคียง)
> - Export เป็น PNG ความละเอียดสูง
> - ห้ามใส่ชื่อแบรนด์เครื่องดื่มจริงใดๆ (ไม่มียี่ห้อ)
> - [วาง PO data จากด้านบน]
